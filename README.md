## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* HeroUI

## Development

Run the project locally:

```bash
pnpm install
pnpm run dev
```

Then open:

```
http://localhost:3000
```

The application will automatically reload when you make changes.

## Build

Create a production build:

```bash
pnpm build
pnpm start
```

## Purpose of the Project

This project is intended solely as a personal resume website to share information about my work experience, projects, and technical background.
